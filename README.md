🛡️ Bait Switch Blaster
An interactive web application designed to teach phishing awareness through game-based learning.

🎯 Objective
Cyber attacks often rely on social engineering rather than technical exploits. This project helps users recognize phishing attempts by presenting realistic scenarios and requiring quick decisions.

🚀 Features
Interactive phishing detection game
Real-world inspired scenarios
Immediate feedback after each response
Gamified learning experience
Designed for classroom workshops
Kid-friendly educational interface
💡 Key Learning Outcome
Users learn that phishing attacks manipulate emotions such as urgency, fear, excitement, or curiosity. The application encourages users to pause, think critically, and verify messages before interacting with them.
